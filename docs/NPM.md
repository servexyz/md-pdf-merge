> ( Multiple markdown files ) => 1 PDF

**Install**

```bash

```

**Execute CLI**

```bash

```

**Test CLI**

## Goal

1. Extract all markdown (`*.md`) files from directory
2. Convert those to PDF
3. Merge those PDFs into one PDF
